# Squint SolidJS Template

## Development

- build: `pnpm run /^build:*/`,
- run dev server: `pnpm run /^dev:*/`,
- preview: `pnpm run preview`
- lint & formatter: WIP

## Libraries

If you need to refer the documents, use context7.

- This project use [Squint](https://github.com/squint-cljs/squint) for `*.cljs`.
- [SolidJS](https://github.com/solidjs/solid) and [solid-router](https://github.com/solidjs/solid-router) for SPA
